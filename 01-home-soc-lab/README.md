# Home SOC Lab

## Project Overview

This project documents the process of building a basic Home SOC Lab for cybersecurity learning and SOC analyst practice.

The lab will include a Windows endpoint monitored by Wazuh and Sysmon, with Kali Linux used to generate test activity.

## Planned Lab Architecture

- Wazuh Server
- Windows 10/11 endpoint
- Sysmon
- Kali Linux attacker machine
- VirtualBox / VMware network

## Project Goals

- Install and configure Wazuh
- Connect a Windows endpoint as an agent
- Install and configure Sysmon
- Collect Windows security logs
- Simulate suspicious activity
- Analyze alerts in Wazuh
- Document findings

## Tools

- Wazuh
- Sysmon
- Windows Event Viewer
- Kali Linux
- Nmap
- PowerShell
