# CV Project Descriptions

## Home SOC Lab

Built a Home SOC Lab using Wazuh, Windows, Sysmon and Kali Linux. Collected endpoint logs, simulated suspicious activity and analyzed alerts from a SOC analyst perspective.

## Cybersecurity Write-ups

Created cybersecurity investigation write-ups covering phishing analysis, Windows Event Logs, Wireshark PCAP analysis and suspicious PowerShell activity.

## Vulnerability Assessment Lab

Performed a vulnerability assessment in a lab environment using Nmap and Nessus/OpenVAS. Identified exposed services, assessed risks and prepared remediation recommendations.
